Patch Notes

See the source code repository for a more thorough, but less readable, change history.

0.1.7 2014/08/31

  • You can now buy multiple upgrades with one click.
  • There are now three buttons for buying units: buy 1, buy 25%, buy max. Buy-exactly-n is gone; it was awkward to use anyway.
  • Removed the buy/select mode from the top. It wasn't widely used, and was confusing if accidentally changed.

0.1.6 2014/08/31

  • More invisible backend changes, nothing to see here.

0.1.5 2014/08/31

  • Unit list is better behaved with long unit names and text selection.
  • Some invisible backend changes.

0.1.4 2014/08/29

  • Mid-tier meat-producing units now warn players who want to start their next tier very early. (High-tier units assume you already know better.)
  • Selecting a unit now shows how fast it's being produced. Statistics page has this too.

0.1.3 2014/08/29

  • Giant Arachnomorphs have a valid price.
  • Any save state with a broken (NaN) value should have it automatically reset to 0.

0.1.2 2014/08/29

  • Number formatting improved for numbers below 999 decillion.

0.1.1 2014/08/28

  • Column buy/buy-max buttons work.

0.1.0 2014/08/28

  • Initial public testing release. Here be dragons.
